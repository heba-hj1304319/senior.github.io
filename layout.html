Abstract

Mobile devices are being used so widely, however, it comes with a major problem which is the small screen size that is not convenient for reading. Therefore, our Android application which act as a prereader, will go through the whole document and give the user a summary with the main ideas in a clear and neat way. It will follow a summarizing algorithm that will rank the sentences relevance. The user can either search for a webpage, or import from mobile memory a document to be summarized. In addition, the summarized text is easier to be converted to speech than a whole document, which will help people with visual aids.
