Why would you like it


Readable Text

Since mobile phones screens are small in size, reading through them are not an easy task. Therefore, our application provides a summary of the selected document in a clear and easy to read font which are listed in bullet points. This will make reading from mobile phones an easier task.


Supports Multiple Languages

Summarize can handle multiple languages such as English, Arabic, and French. Since the used summarization algorithm does not depend on language semantics, it can be used for multiple languages.

Convenient for Researchers

Researchers who are looking for resources on the web for their research and need to access multiple websites in a limited time can use our application to pre-read the website and then if he/she like it, then read the whole document. Summarize will reduce time and effort needed to accomplish this task. The application can help all types of researchers such as school students, college students, as well as people who seek knowledge from websites.
