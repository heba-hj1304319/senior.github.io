What is Summarize


The Application

Summarize is an Android application that can act as a pre-reader for the user who is using his/her mobile phone to read some documents. The application can have 2 main sources of the documents:

  1. It can provide a google search functionality to surf through the web and select a document from the search results.
  2. You can import an existing document from your mobile device memory to our app to be summarized.
In addition, the summarized text can be saved to a database and retrieved later whenever needed.

How does it work

To summarize a document, we are using Hyper Rectangular Algorithm which is based on Formal Concept Analysis. This algorithm can obtain the most relevant sentences in a document, which can be represented as a summary.

The basic steps that can be done to get the most important sentences.


The Incremental Approach

Summarizing the whole document all at once might take a long time, especially if the document contains large amount of text. In the incremental approach, after the document has gone through the original algorithm, the rest of the documents will be divided into chunks. Each time a new chunk will undergo the algorithm with the previously computed summary by using the already calculated weights from the previous chunks.
